# 感知机模式

感知机会计算输入的加权和，再加上偏置，然后应用阶跃函数，输出一个二分类结果。它是神经网络的基本单元。

```text
output = step(w1*x1 + w2*x2 + ... + wn*xn + bias)
```

## 什么时候单个感知机就够了

- 问题是线性可分的：一条直线或一个超平面可以分开两个类别
- 逻辑门：AND、OR、NOT、NAND
- 简单阈值决策：例如“分数是否高于 X？”
- 数据聚成两个互不重叠区域的二分类问题

## 什么时候需要多层

- 问题不是线性可分的：没有单条线能分开类别
- XOR 和奇偶校验问题
- 任何需要“这个条件成立，但那个条件不成立”的组合推理任务
- 真实世界分类：图像、文本、音频几乎总是非线性的

## 决策清单

1. 画出或检查你的数据。类别之间能否用单个直线边界分开？
   - 可以：单个感知机可用
   - 不可以：至少需要两层
2. 问题能否拆成更简单线性决策的 AND 或 OR 组合？
   - 这种拆解会告诉你最小网络结构
   - `XOR = (A OR B) AND (NOT (A AND B))`，也就是 2 层中的 3 个感知机
3. 对于多分类问题，每个类别需要一个输出节点

## 训练规则

```text
error = expected - predicted
weight_new = weight_old + learning_rate * error * input
bias_new = bias_old + learning_rate * error
```

如果预测正确，参数不变。如果预测错误，权重会朝着减少错误的方向移动。这个规则只适用于单层感知机。多层网络需要反向传播。

## 常见错误

- 试图用单个感知机学习非线性模式，它永远不会收敛
- 学习率设得太高导致权重震荡，或太低导致训练极慢
- 忘记偏置项；没有偏置时，决策边界必须穿过原点
- 混淆感知机收敛和一般神经网络收敛：前者在线性可分数据上有保证，后者没有通用保证
