# 理解力检测

测试你在「AI 从零开始」课程中某个已完成阶段的知识掌握程度。

## 激活条件

当用户说出以下内容时激活此技能：
- `/check-understanding 3` 或 `/check-understanding deep-learning`
- "考考我阶段 2"
- "测试阶段 1"
- "检查我对 Transformer 的理解"
- "我掌握了阶段 3 吗"
- "我准备好进入下一阶段了吗"

## 输入

接受阶段编号（0-19）或阶段名称作为参数。如果未提供参数，列出全部 20 个阶段供用户选择。

## 流程

### 步骤 1：解析阶段

解析参数。如果是数字，验证范围在 0 到 19 之间（含边界）。超出范围则提示用户："阶段 [N] 不存在。有效阶段为 0-19。" 并显示完整列表供选择。如果是名称或关键词，在上方阶段映射表中查找。如果未匹配任何条目，提示用户："未知阶段 '[关键词]'。请从下方列表中选择：" 并展示全部 20 个阶段。如果未提供参数，要求用户从完整列表中选择。

### 步骤 2：读取阶段内容

使用 Glob 查找 `phases/<阶段目录>/` 下的所有课时目录。对每个课时，读取 `docs/en.md` 文件。这些文档包含用于生成题目的教学内容。

尽可能多读课时文档以覆盖该阶段的完整知识面。如果某阶段课时较多（15+），优先选取有代表性的分布：前几课、中间几课和最后几课。

### 步骤 3：生成 8 道题目

根据刚才读取的课时内容，创建恰好 8 道选择题：

**第 1-4 题：概念题（是什么/为什么）**
测试对概念、定义和推理的理解。示例：
- "X 的目的是什么？"
- "当 Z 发生时，为什么会出现 Y？"
- "以下哪个选项最准确地描述了 A 和 B 之间的关系？"
- "X 解决了什么问题？"

**第 5-8 题：实践题（怎么做/如何构建）**
测试应用知识和实现能力。示例：
- "你会如何实现 X？"
- "哪种方法能正确解决 Y？"
- "构建 Z 的正确步骤顺序是什么？"
- "如果训练过程中观察到 X，你应该怎么做？"

每道题必须有 3 或 4 个选项，标记为 A、B、C（可选 D）。有且仅有一个正确选项。错误选项应具有迷惑性，但对学过该内容的人来说明显不正确。

每道题标注其来源课时（例如："课时 03：矩阵变换"）。

### 步骤 4：逐题展示

使用 **AskUserQuestion** 工具（或等效交互式提示）逐题展示。格式：

```
第 1/8 题（概念题）—— 来自课时 03：矩阵变换

特征值的几何含义是什么？

A) 矩阵施加的旋转角度
B) 变换过程中特征向量被缩放的倍数
C) 变换矩阵的行列式
D) 变换后矩阵的秩
```

等待用户作答后再进入下一题。

### 步骤 5：记录与评分

保持动态计分：
- 总正确数（满分 8）
- 对每道错误的题目，记录：题号、用户答案、正确答案以及来源课时

### 步骤 6：展示结果

全部 8 题完成后，显示得分和等级：

**7-8 题正确：已掌握**
如果是阶段 19（综合实战项目）："你已经掌握了最后一个阶段。恭喜你完成了整个课程！"
否则："你对阶段 N 掌握扎实，可以继续学习阶段 N+1：[下一阶段名称]。"

**5-6 题正确：接近掌握**
"基础扎实。继续学习前建议复习以下内容："
然后列出与错误题目相关的课时。

**3-4 题正确：发展中**
"你正在建立理解，但需要重新学习部分课时："
然后列出每道错题及对应的课时。

**0-2 题正确：需要重学**
"这个阶段需要更多时间。请从头重新学习，重点关注："
然后列出所有未掌握的主题。

### 步骤 7：错题解析

对每道答错的题目，显示：

```
第 N 题：[题目内容，简要]
你的答案：B
正确答案：C —— [正确选项内容]
原因：[1-2 句解释为什么 C 是正确的]
复习：课时 NN —— [课时名称]（phases/<阶段目录>/NN-<课时slug>/docs/en.md）
```

### 步骤 8：接下来做什么？

最后提供三个选项：

1. **重新测试** —— 从同一阶段生成全新的 8 道题
2. **换一个阶段** —— 选择另一个阶段进行测试
3. **讲解知识点** —— 询问答错题目中的任何概念

等待用户选择并相应执行。

## 规则

- 重新测试时避免重复已出过的题目，直到题库用尽。用尽后打乱顺序或重新措辞。
- 题目必须直接源于课时文档，不得使用通用知识。
- 在用户作答前不得显示正确答案。
- 题目文本保持简洁，最多一到两句话。
- 错误选项必须具有迷惑性，不得使用明显荒谬的答案。
- 如果某阶段尚无课时内容（未找到 `en.md` 文件），提示用户："阶段 N 尚无课时内容。请选择一个已完成的阶段进行测试。"
